Browsers

WorldClassRoom is a cloud-based system run by Instructure Canvas. So there is no software that you need to install on your computer to run your online course. Canvas supports all modern web browsers. Canvas supports the latest two versions of any given browser.

Review the latest browser requirements in the Canvas Guides for the latest information.

It is important to keep the browser up-to-date on any computer you will be using to teach your class. We do recommend that you have at least two different browsers installed on your computer in case there are technical issues.Firefox    Chrome    Safari       Edge

Mobile Support

Webster University does not have the resources to support all mobile devices that are available to instructors and students. While Canvas works fairly well on mobile devices (phones and tablets), there may be some things that do not work as well as they do on desktop web browser so we recommend that you do most of your instructing on a laptop or desktop. As with any app, not all features will be as available as they will be on a regular computer.

Canvas is built for web standards and with accessibility in mind and is accessible via an App or mobile browser.

Troubleshooting

You are not responsible for all of the technical issues that students encounter. The general rule of thumb is this: If everyone in the class is having a technical issue with, say a video, then the issue is probably on our (the OLC) part and you should contact us or the IT Service desk to see if we can remedy the problem. If only individual students are having issues, then you should send them to the IT Service desk, where there are people trained to troubleshoot technical issues on individual computers.
