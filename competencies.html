Module 1

Competencies expected of Online Instructors

Technical Competencies

All Webster University online instructors should be able to:

      1. Use the Inbox for communication
      2. Create Canvas Announcements
      3. Modify assignment details, dates, and settings
      4. Launch a quiz with acceptable settings
        • Add extra time for individual students
      5. Grade assignments using Canvas tools
        • Annotate documents in speedgrader
        • Grade with a rubric, if desired
      6. Use discussions for engaging students
        • Modify description and dates
        • Reply to student postings
      7. Monitor student progress using
        • People page
        • Gradebook
      8. Use video recording tools
      9. Understand how to make course materials accessible
      10. Navigate using a modern browser
      11. Upload files for student access
      12. Fix broken links
        • Find with Link Validator

Affective Competencies

All Webster University Online instructors must have:

    1. Willingness to:
        • Learn about teaching and learning in an online format
        • Be the "guide on the side"
        • Understand of how people learn
        • Update and engage in new practices
    2. Openness to learning
    3. Attentiveness to the unique challenges of distance learning

Organizational Competencies

All Webster University instructors must be able to:

  1. Use the tools of O365 make aware
  2. Use Connections
      • Access Class list
      • Submit final grades
      • Locate FaST site
      • Access information about pay schedules
  3. Know the academic calendar
  4. Understand and respect FERPA

Pedagogical Competencies

All Webster University online instructors should:

  1. Be a subject matter expert in your field
  2. Achieve mastery of provided course content, structure, and organization
  3. Respond to student inquiries in a timely fashion
  4. Communicate course goals
  5. Provide timely, meaningful, and consistent feedback to individuals and groups
  6. Use student data (e.g. submissions and communications) to guide the feedback process
  7. Provide information to students about their progress using system tools
  8. Provide feedback in written, audio, and video forms
